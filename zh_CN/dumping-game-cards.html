# Dumping Game Cards

This section is for dumping Game Cards using GodMode9i so they can be played on emulators, flashcards, or your SD card via nds-bootstrap.

# 需要准备

  • The Game Card of the title you would like to dump
  • 一台安装了Unlaunch的Nintendo/iQue DSi

# 操作步骤

# 第一节 - SD 卡设置

TIP

正在使用Windows、Linux或 macOS 设备？ 使用 Lazy DSi Downloader 来自动设置您的 SD 卡。

  1. 下载最新版本的 GodMode9iopen in new window
  2. 从 GodMode9i 目录中解压 GodMode9i.nds 并复制到你SD卡的任意位置

# 第二节——DSi上的操作

  1. 摁住 A 和 B键 后，再摁下 Dsi 的电源键
    • 这应该能启动到Unlaunch Filemenu
  2. 从 Unlaunping Filemenu 启动 GodMode9i
  3. Ensure the Game Card is inserted into the console
  4. 选择 GodMode9i 中的“NDS GAMECARD”选项
  5. 选择你是否需要空白填充内容（无游戏数据的内容）：
    • Full: 包括填充内容
    • Trim: 移除填充内容
  6. Repeat steps 3-5 for all Game Cards you wish to dump

TIP

The dumped Game Cards will be found in sd:/gm9i/out.
