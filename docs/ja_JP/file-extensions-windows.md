# ファイル拡張子(Windows)

この補足セクションでは、ファイル拡張子を隠すWindowsのデフォルト動作を無効にします。 ガイド内で示されるファイルを見つけるのが簡単になります。

::::: tabs

:::: tab default Windows 11

1. エクスプローラでSDカード等の任意のディレクトリを開きます
2. トップバーの「表示」オプションをクリックします
   - オプションがない場合は、トップバーの右端にある`···`ボタンをクリックしてください
3. 表示 サブメニューにカーソルを合わせるかクリックします
4. `ファイル名拡張子`というオプションにチェックを入れてください
   ![Windows 11 で「ファイル名拡張子」チェックボックスにマウスを合わせたスクリーンショット](/assets/images/windows-11-file-extensions.png)

::::

:::: tab Windows 10 / 8.1 / 8

1. エクスプローラでSDカード等の任意のディレクトリを開きます
2. リボンメニューの「表示」オプションをクリックします
3. `ファイル名拡張子`のチェックボックスにチェックを入れてください
   ![Windows 10 で「ファイル名拡張子」チェックボックスにマウスオーバーしたスクリーンショット](/assets/images/windows-10-file-extensions.png)

::::

:::: tab Windows 7 / Vista / XP

1. クリックするかWindowsキーを押して、スタートメニューを開きます
2. 「フォルダー オプション」を検索し、該当する結果を選択してください
   ![Windows 7 スタートメニューで「フォルダー オプション」を検索したスクリーンショット](/assets/images/windows-7-folder-options-start-menu.png)
3. フォルダー オプション メニューの上部にある「表示」オプションをクリックします
4. `登録されている拡張子は表示しない`というチェックボックスがオフになっていることを確認してください
   ![Windows 7 の「登録されている拡張子は表示しない」がオフになっている場合の「フォルダー オプション」ウィンドウのスクリーンショット](/assets/images/windows-7-folder-options.png)
