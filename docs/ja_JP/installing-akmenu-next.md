# AKMenu-Next のインストール

1. [AKMenu-Next DSi Edition](https://github.com/coderkei/akmenu-next/releases/latest/download/akmenu-next-dsi.zip) の最新版をダウンロードしてください
2. `akmenu-next-dsi.zip`を解凍してください
3. If Unlaunch is not installed, delete `boot.nds` from the root of your SD card
4. AKMenu-Nextのファイル内から、次のファイル/フォルダをSDカードのルートにコピーしてください:
   - `_nds` フォルダ
   - `BOOT.NDS`
   - `akmenu-next.dsi`
   - If hiyaCFW is installed, copy the `title` folder to your SD card root as well
5. 最新リリースの [nds-bootstrap](https://github.com/DS-Homebrew/nds-bootstrap/releases/latest/download/nds-bootstrap.zip) をダウンロードしてください
6. `nds-bootstrap.zip`を解凍してください
7. 解凍した`nds-bootstrap.zip`ファイル&#x306E;_&#x5185;&#x5BB9;_&#x3092;SDカードの`_nds`フォルダにコピーしてください

::: tip

If you want to play games from before the DS(i) (such as GBA and below) and/or use other additional features (known as plugins), see [Installing Plugins](https://coderkei.github.io/akmenu-next-docs/guides/plugins/).

:::

### インストール後

dumpTool の起動に[使用したエクスプロイト](launching-the-exploit.html)を実行すると、AKMenu-Next が起動します。
