## 홈브류란 무엇입니까?

[홈브류](https://en.wikipedia.org/wiki/Homebrew_(video_games))란 닌텐도 DSi와 같이 폐쇄형 시스템을 위해 만든 라이센스 없는 프로그램을 뜻합니다. 이러한 응용 프로그램은 유틸리티에서 커스텀 홈브류 게임에 이르기까지 다양합니다.

홈브류는 펌웨어 버전이나 지역에 관계없이 모든 닌텐도 DSi 기기에서 무료로 실행할 수 있습니다. 필요한 자료들은 진입 지점하고 홈브류를 보관할 SD 카드입니다. 이 가이드는 Memory Pit라는 진입 지점을 사용하지만, 이 방법이 불가능하다면 다른 진입 지점을 사용 할 수 있습니다.

## 시스템을 개조하여 무엇을 할 수 있나요?

- 플래시카드 없이 DSi SD카드에서 닌텐도 DS(i) 게임 백업파일이나 ROM 핵을 실행할 수 있음
- SD카드에서 DSiWare 프로그램을 실행할 수 있음
  - 다시 말해, 지역 락이 걸려있거나 3DS에서만 판매하던 DSiWare도 가능함
- 닌텐도 DSi를 켜는 중에 특정 버튼을 누르고 있으면 DSiWare나 특정 홈브류 앱으로 부팅 가능
- 다양한 에뮬레이터를 이용해 오래된 클래식 게임들 실행 가능
- 일반적으로 호환되지 않는 플래시 카드 사용 가능
- hiyaCFW를 통해 NAND를 SD 카드로 리다이렉션 가능
- FastVideoDSPlayer나 tuna-viDS를 이용해 좋아하는 영화를 감상 가능
  - FastVideoDSEncoder는 AVX2를 지원하는 CPU가 필요함 (일반적으로 최신 CPU는 대부분 지원)
- 시스템 시작 시 이미지(부팅 스플래쉬라고 함) 표시
- 홈브류 게임 플레이 가능

## 홈브류 앱을 어디서 찾을 수 있나요?

- [Universal-DB](https://db.universal-team.net/ds)를 통해 대부분의 최신 홈브류를 검색해보고 정렬해볼수 있으며, 최근 업데이트 내역을 살펴볼 수 있습니다
- [GameBrew](https://www.gamebrew.org/wiki/List_of_all_DS_homebrew)에는 많은 오래된 홈브류가 있으며, 몇몇은 플래시카드에서만 동작합니다

## 시작하기 전에 알아야 할 것은 무엇인가요?

- Windows에서 기본 파일 탐색기를 사용하는 경우 [파일 확장자명 표시](file-extensions-windows.html)를 권장합니다.
- 유일하게 벽돌이 될 가능성은 Unlaunch를 설치하는 데서 발생하며 벽돌이 될 가능성은 극히 낮습니다
