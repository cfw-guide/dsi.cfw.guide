# DSiWare 백업하기

## 준비물

- 최신 버전의 [GodMode9i](https://github.com/RocketRobz/godmode9i/releases)
  - 아카이브를 다운로드하고 콘텐츠를 추출한 다음 SD 카드의 아무 곳에나 `GodMode9i.nds`를 배치합니다.

## Nintendo DSi - 지침

### 섹션 I - DSiWare 덤프

1. GodMode9i 실행
2. <kbd>START</kbd>를 눌러 시작 메뉴 열기
3. `Title manager...` 선택
   - 만약 보이지 않는다면, SD card와 NAND가 마운트 되었는지 확인하세요. hiyaCFW에서 로딩하는 경우 다른 곳에서 다시 로딩하세요
4. 덤프하고 싶은 타이틀을 선택해 주세요
5. 덤프하고 싶은 것을 선택해 주세요
6. 덤프하고 싶은 모든 DSiWare 들을 4-5를 반복하며 진행해 주세요

::: tip

덤프된 DSiWare는 `sd:/gm9i/out` 에서 찾을 수 있습니다.

:::
