# 게임 카드 덤핑하기

이 문서는 GodMode9i를 이용해 에뮬레이터, 플래시카드, nds-bootstrap 등에서 사용 가능한 게임 파일로 덤핑하는 방법에 대해 다룹니다.

## 준비물

- 덤프하고 싶은 게임 카드
- [Unlaunch](installing-unlaunch.html)가 설치된 닌텐도 DSi 콘솔

## 사용 설명

### 섹션 I - SD 카드 설정

1. 최신 버전의 [GodMode9i](https://github.com/DS-Homebrew/GodMode9i/releases)를 다운로드 합니다
2. `GodMode9i.nds`파일을 GodMode9i 압축파일에서 추출한 뒤, SD카드 아무곳에나 넣으세요

### 섹션 II - 닌텐도 DSi 설정

1. GodMode9i를 실행합니다
2. 게임 카드를 콘솔에 꽂아 주세요
3. GodMode9i에서 "NDS GAMECARD" 옵션을 선택해 주세요
4. 덤프하고 싶은 것을 선택해 주세요
   - ROM 덤프 시 "Trimmed" 옵션은 용량을 더 작게 차지하지만, ROM 핵과 같은 대부분의 패치가 작동하지 않을 겁니다
5. 덤프하고 싶은 모든 게임카드들을 2-4를 반복하며 진행해 주세요

::: tip

덤프된 게임 카드는 `sd:/gm9i/out` 에서 찾을 수 있습니다.

:::
