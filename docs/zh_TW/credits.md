# 致謝

這是對那些為指南網站、自製軟體和其他事情提供幫助的人的致謝。

如果您喜歡他們所做的事情，可以考慮捐款（如果他們有捐款連結）。

- [NightScript](https://nightscript370.github.io/), [Pk11](https://pk11.sh/) & [Peter0x44](https://github.com/Peter0x44): Guide maintainers
- [emiyl](https://emiyl.com/paypal): 域名持有人
- Halvorsen: 提供一張 Nintendo DSi 的圖片用於首頁
- [shutterbug2000](https://paypal.me/projectkaeru): Nintendo DSi 照相館和 Flipnote Studio 漏洞的發現者和開發者
- [nathanfarlow](https://github.com/nathanfarlow): Nintendo DSi 瀏覽器漏洞的發現者和開發者
- [WinterMute](https://devkitpro.org/support-devkitpro): DS Homebrew 開發工具鏈的維護者，並分支了 ugopwn 的 Flipnote Studio 漏洞以製作 Flipnote Lenny（支援所有系統版本和地區）
- [nocash](http://problemkaputt.de/donate.htm): Unlaunch 的開發者
- [RocketRobz](https://github.com/RocketRobz): TWiLight Menu++ 和 GodMode9i 的開發者，以及 hiyaCFW 的貢獻者
- [Kai](https://github.com/coderkei): Developer of AKMenu-Next and writer of the AKMenu-Next installation steps from [AKMenu-Next Documentation](https://coderkei.github.io/akmenu-next-docs/guides/akmenu-next/#__tabbed_1_3)
- [zoogie](https://github.com/zoogie): dumpTool 的開發者，並分支了 Memory Pit 以支援所有地區和韌體
- [Plailect](https://github.com/Plailect): 製作了原版 Nintendo 3DS 破解指南，此指南的靈感

您也可以透過 [GitHub 上傳送請求](https://github.com/cfw-guide/dsi.cfw.guide/)為我們的指南做出貢獻！

如果您懂其他語言，您可以幫助我們將指南翻譯成其他語言。 您可以加入我們的 [Crowdin 專案](https://crowdin.com/project/dsi-guide)來開始使用。
