# 安裝 TWiLight Menu++

1. 下載最新版本的[`TWiLightMenu-DSi.7z`](https://github.com/DS-Homebrew/TWiLightMenu/releases/latest/download/TWiLightMenu-DSi.7z)
   - 如果無法下載，請參閱[發布頁面](https://github.com/DS-Homebrew/TWiLightMenu/releases/latest)並向下捲動至`Assets`下載檔案
2. 解壓縮`TWiLightMenu-DSi.7z`
   - If you're using Windows, right-click on `TWiLightMenu-DSi.7z`, click `Show more options` (if using Win11), hover over `7-zip`, then click `Open archive`
3. If Unlaunch is not installed, delete `boot.nds` from the root of your SD card
4. From `TWiLightMenu-DSi`, copy the following files/folders to your SD card root:
   - `_nds`
   - `BOOT.NDS`
   - `roms`
   - If hiyaCFW is installed, copy the `title` folder to your SD card root as well

::: tip

如果你想玩 Nintendo DS(i) 和 Game Boy Advance 之前的遊戲和/或使用其他附加功能（稱為附加元件），請參閱[安裝附加元件](https://wiki.ds-homebrew.com/twilightmenu/installing-addons?tab=manual)。

:::

### 完成安裝後

執行您[使用的漏洞程式](launching-the-exploit.html)來啟動 dumpTool，然後 TWiLight Menu++ 就會啟動。 有可能會要求您選擇語言和地區。 這些選項無需與您的主機語言或地區匹配，您可以根據自己的喜好進行設定。

::: tip

繼續至[安裝 Unlaunch](installing-unlaunch.html) (非必需)

:::
