# 檔案副檔名 (Windows)

這是一個附加部分，用於停用 Windows 隱藏檔案副檔名的預設行為，這將使您更容易在指南中找到引用的檔案。

::::: tabs

:::: tab default Windows 11

1. 啟動檔案總管，前往到任何資料夾/目錄（例如：SD卡根目錄）
2. 點觸工具列上的「檢視」選項
   - 如果沒有此選項，點處工具列最右側的「···」按鈕
3. 按一下或將遊標懸停在`顯示 >`子選單上
4. 勾選「副檔名」選項
   ![Windows 11 上懸停「副檔名」複選框的螢幕截圖](/assets/images/windows-11-file-extensions.png)

::::

:::: tab Windows 10 / 8.1 / 8

1. 啟動檔案總管，前往到任何資料夾/目錄（例如：SD卡根目錄）
2. 點觸功能區上的「檢視」選項
3. 勾選「副檔名」選項
   ![Windows 10 上懸停「副檔名」複選框的螢幕截圖](/assets/images/windows-10-file-extensions.png)

::::

:::: tab Windows 7 / Vista / XP

1. 按一下或使用 Windows 鍵開啟開始選單
2. 搜尋「資料夾選項」並選擇相應的結果
   ![Windows 7 開始選單中搜尋「資料夾選項」的螢幕截圖](/assets/images/windows-7-folder-options-start-menu.png)
3. 點觸資料夾選項的「檢視」標籤
4. 請確認未選取「隱藏已知檔案類型的副檔名」複選框。
   ![Windows 7 系統中「資料夾選項」視窗的螢幕截圖，其中「隱藏已知類型的副檔名」已關閉](/assets/images/windows-7-folder-options.png)
