## 什麼是自製軟體？

[自製](https://en.wikipedia.org/wiki/Homebrew_(video_games))軟體是未經授權的軟體，專為 Nintendo DSi 等封閉主機開發。 這些應用程式的範圍從工具程式到自製遊戲。

自製軟體可以在所有 Nintendo DSi 主機上免費執行，無論主機版本或區域如何。 您所需要的只是一個入口點和一張SD卡來儲存您的自製軟體。 此指南中使用的主要入口點稱為 Memory Pit，但如果 Memory Pit 無法使用，您也可以使用其他入口點。

## 破解我的主機之後能做什麼？

- 即可從 Nintendo DSi 的SD卡執行 Nintendo DS(i) 遊戲備份或 ROM hack，無需燒錄卡
- 從SD卡啟動任何 DSiWare
  - 不受區域限制且 Nintendo 3DS 專屬的 DSiWare 也可以使用
- 開啟 Nintendo DSi 主機電源時按住特定按鈕即可啟動 DSiWare 和自製軟體
- 使用各種模擬器執行經典舊遊戲
- 使用不相容的燒錄卡
- 使用 hiyaCFW 將系統NAND重新導向到SD卡
- 使用 FastVideoDSPlayer 或 tuna-viDS 觀看您喜愛的電影
  - FastVideoDS 編碼器需要處理器(CPU)支援AVX2（較新的處理器可能會支援它）
- 在主機啟動時顯示圖像（稱為啟動畫面）
- 玩自製遊戲

## 從哪裡可以找到自製軟體？

- [Universal-DB](https://db.universal-team.net/ds) 擁有最現代化的自製軟體，並具有良好的搜尋和排序功能，可以追蹤最近的更新內容
- [GameBrew](https://www.gamebrew.org/wiki/List_of_all_DS_homebrew) 有很多較舊的自製軟體，但大部分只能在燒錄卡上執行

## 在開始之前我應該要知道什麼？

- 如果您使用 Windows 檔案總管，建議[顯示檔案副檔名](file-extensions-windows.html)
- 唯一的當機風險來自於安裝 Unlaunch，而且當機風險很小
