# DSiWare 備份

## 需求

- 最新版本的 [GodMode9i](https://github.com/RocketRobz/godmode9i/releases)
  - 下載壓縮檔案，解壓縮內容並將 `GodMode9i.nds` 放在SD卡上的任何位置

## Nintendo DSi - 使用說明

### 第一部分 - 匯出 DSiWare

1. 啟動 GodMode9i
2. 按<kbd>START</kbd>開啟START選單
3. 選擇「Title manager...」
   - 如果沒有顯示，請確認SD卡和NAND都已掛載。 如果是從 hiyaCFW 載入，請從其他方式重新載入
4. 選擇想要匯出的軟體
5. 選擇想要匯出的內容
6. 對您想要匯出的所有 DSiWare 軟體，重複第4至5步驟

::: tip

已匯出的 DSiWare 將在`sd:/gm9i/out`中可以找到。

:::
