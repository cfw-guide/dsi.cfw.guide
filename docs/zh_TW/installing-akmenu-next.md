# 安裝 AKMenu-Next

1. 下載最新版本的 [AKMenu-Next DSi Edition](https://github.com/coderkei/akmenu-next/releases/latest/download/akmenu-next-dsi.zip)
2. 解壓縮`akmenu-next-dsi.zip`
3. If Unlaunch is not installed, delete `boot.nds` from the root of your SD card
4. 從 AKMenu-Next 檔案中，將這些檔案/資料夾複製到SD卡根目錄：
   - `_nds`資料夾
   - `BOOT.NDS`
   - `akmenu-next.dsi`
   - If hiyaCFW is installed, copy the `title` folder to your SD card root as well
5. 下載最新版本的 [nds-bootstrap](https://github.com/DS-Homebrew/nds-bootstrap/releases/latest/download/nds-bootstrap.zip)
6. 解壓縮`nds-bootstrap.zip`
7. 將解壓縮後的`nds-bootstrap.zip`檔案中的檔&#x6848;_&#x5167;&#x5BB9;_&#x8907;製到SD卡上的`_nds`資料夾中

::: tip

If you want to play games from before the DS(i) (such as GBA and below) and/or use other additional features (known as plugins), see [Installing Plugins](https://coderkei.github.io/akmenu-next-docs/guides/plugins/).

:::

### 完成安裝後

執行您[使用的漏洞程式](launching-the-exploit.html)來啟動 dumpTool，然後 AKMenu-Next 就會啟動。

::: tip

繼續至[安裝 Unlaunch](installing-unlaunch.html) (非必需)

:::
