# Desinstalando o Unlaunch

::: danger

Installing or uninstalling Unlaunch, while safe, writes to the console's NAND, so there's a small chance to brick your console!

:::

**WARNING:** While uninstalling Unlaunch is relatively safe, there is still a small chance of bricking your Nintendo DSi. Aqui estão alguns casos sobre por que você pode querer desinstalar o Unlaunch mas com soluções que não exigem desinstalação.

- **I don't want to use a custom menu anymore and I don't like having the Unlaunch file menu show on boot:** You can follow the [Unlaunch post install steps](installing-unlaunch.html#section-iv-post-unlaunch-configuration) to have Unlaunch autoboot the DSi menu
- **The Unlaunch Background is scary:** [Reinstall Unlaunch](installing-unlaunch.html) by selecting one of the available custom backgrounds
- **Eu estou tendo um problema com o Unlaunch ou meu console após instalá-lo:** A página [Troubleshooting](troubleshooting.html#unlaunch) explicará como corrigir muitos problemas que você pode ter

::: warning

Para Reduzir as chances de bricking, certifique-se de que você não instalou nenhum DSiWare ilegítimo em sua NAND backup (o redirecionamento fornecido pela hiyaCFW não conta), ou alterou de outra forma os arquivos do sistema.

:::

Once you have reviewed the above information, follow the [Installing Unlaunch](installing-unlaunch.html) instructions up to the uninstall part.
