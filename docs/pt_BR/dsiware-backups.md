# Backups de DSiWare

## Requisitos

- A última versão do [GodMode9i](https://github.com/RocketRobz/godmode9i/releases)
  - Baixe o arquivo, extraia os conteúdos e coloque `GodMode9i.nds` em qualquer lugar do seu cartão SD

## Nintendo DSi - Instruções

### Seção I - Extraindo o DSiWare

1. Inicie o GodMode9i
2. Pressione <kbd>START</kbd> para abrir o menu Iniciar
3. Selecione `Title manager...`
   - Se essa opção não aparecer, certifique-se de que você tem seu cartão SD e NAND montados. Se você estiver usando o hiyaCFW, reinicie e abra o GodMode9i de algum outro lugar
4. Selecione o título que você quer fazer um dump
5. Selecione o que você quer fazer um dump
6. Repita os passos 4-5 para todos os DSiWare que você deseja extrair

::: tip

O dump do DSiWare será encontrado em `sd:/gm9i/out`.

:::
