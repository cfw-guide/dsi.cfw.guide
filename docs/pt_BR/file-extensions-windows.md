# Extensões de arquivos (Windows)

Esta é uma seção adicional para desabilitar o comportamento padrão do Windows de esconder extensões de arquivo, o que facilitará muito a busca de arquivos referenciados no guia.

::::: tabs

:::: tab default Windows 11

1. Inicie o Explorador de Arquivos abrindo qualquer pasta, como a pasta do seu cartão SD
2. Clique na opção "Exibir" na barra superior
   - Se a opção não estiver presente, clique no botão ·· na parte superior direita da barra superior
3. Clique ou coloque seu cursor sobre o submenu Mostrar
4. Verifique a opção rotulada `extensões de nome de arquivo`
   ![Screenshot of hover the "File name extensions" checkbox in Windows 11](/assets/images/windows-11-file-extensions.png)

::::

:::: tab Windows 10 / 8.1 / 8

1. Inicie o Explorador de Arquivos abrindo qualquer pasta, como a pasta do seu cartão SD
2. Clique na aba "Exibir" na faixa de opções superior
3. Verifique a opção rotulada `extensões de nome de arquivo`
   ![Screenshot of hover the "File name extensions" checkbox in Windows 10](/assets/images/windows-10-file-extensions.png)

::::

:::: tab Windows 7 / Vista / XP

1. Abra o menu iniciar ao clicar nele ou na tecla Windows
2. Pesquise por "Opções de pasta" e selecione o resultado correspondente
   ![Captura de uma busca por "opções de pasta" no Windows 7 Iniciar Menu](/assets/images/windows-7-folder-options-start-menu.png)
3. Clique na opção "Modo de Exibição" na parte superior do menu de Opções de pasta
4. Certifique-se de que a caixa "Ocultar extensões para tipos de arquivos conhecidos" está desmarcada como
   ! Captura de tela da janela "Opções de Pasta" no Windows 7 com "Ocultar extensões para tipos conhecidos" desligada](/assets/images/windows-7-folder-options.png)
