## O que é homebrew?

[Homebrew](https://en.wikipedia.org/wiki/Homebrew_(video_games)) aplicações são software não licenciado feito para sistemas fechados como o Nintendo DSi. Esses aplicativos podem ir de utilitários a jogos homebrew personalizados.

O Homebrew pode ser rodado gratuitamente em todos os consoles do Nintendo DSi, não dependendo da versão do firmware ou região. Tudo que você precisa é de um ponto de entrada e um cartão SD para armazenar seu homebrew. O ponto de entrada principal usado neste guia é chamado Memory Pit, mas existem outros pontos de entrada que você pode usar se o Memory Pit estiver inutilizável.

## O que posso fazer modificando o meu sistema?

- Rodar backups de jogos de Nintendo DS(i) ou ROM hacks pelo seu cartão SD do DSi sem a necessidade de um flashcard
- Iniciar qualquer DSiWare do seu cartão SD
  - Isto significa que DSiWares de outras regiões ou exclusivos do 3DS também funcionarão
- Iniciar DSiWare e aplicativos homebrew segurando botões específicos ao ligar seu Nintendo DSi
- Rodar jogos clássicos usando vários emuladores
- Usar flashcards normalmente incompatíveis
- Redirecionar sua NAND para o cartão SD usando o hiyaCFW
- Assistir os seus filmes favoritos usando o FastVideoDSPlayer ou tuna-viDS
  - FastVideoDSEncoder precisa de uma CPU suportada por AVX2 (CPUs mais recentes provavelmente suportarão)
- Exibir uma imagem (referida como o boot splash) ao iniciar o sistema
- Jogar jogos homebrew

## Onde posso encontrar aplicativos homebrew?

- A <a href="https://db.universal-team.net/ds">Universal-DB</a> tem a maioria dos homebrews modernos, com uma bom sistema de busca e ordenação para acompanhar o que foi atualizado recentemente
- [GameBrew](https://www.gamebrew.org/wiki/List_of_all_DS_homebrew) tem um monte de homebrews antigos, porém muitos só funcionarão com flashcards

## O que eu preciso saber antes de começar?

- No Windows, é recomendado mostrar extensões de arquivos se você estiver usando o gerenciador de arquivos padrão
- O único risco de brick vem com a instalação do Unlaunch, e o risco do brick é mínimo
