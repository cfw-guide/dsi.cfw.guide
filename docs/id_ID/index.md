## Apa itu homebrew?

[_Homebrew_](https://en.wikipedia.org/wiki/Homebrew_\(video_games\)) adalah aplikasi tanpa lisensi yang dibuat untuk sistem tertutup seperti Nintendo DSi. Aplikasi yang ada mulai dari alat kegunaan sampai permainan _homebrew_.

_Homebrew_ bisa dijalankan bebas di semua Nintendo DSi, terlepas dari versi _firmware_ atau daerah konsol. Hanya perlu titik masuk dan kartu SD untuk menyimpan _homebrew_. Titik masuk utama pada panduan ini menggunakan Memory Pit, tapi ada juga titik masuk yang lain jika Memory Pit tidak bisa.

## Bisa apa saja setelah memodif sistem?

- Menjalankan cadangan permainan Nintendo DS(i) atau retasan ROM dari kartu SD DSi tanpa _flashcard_
- Menjalankan DSiWare apa pun dari kartu SD
  - Ini berarti DSiWare yang luar-daerah dan eksklusif-3DS juga bisa
- Membuka DSiWare atau _homebrew_ dengan menahan tombol spesifik saat menyalakan Nintendo DSi
- Bermain permainan klasik dengan beragam emulator
- Menggunakan _flashcard_ yang biasanya tidak bisa
- Mengarahkan bacaan NAND ke kartu SD dengan hiyaCFW
- Menonton film kesukaan dengan FastVideoDSPlayer atau tuna-viDS
  - FastVideoDSEncoder perlu CPU komputer yang bisa AVX2 (CPU terbaru mungkin bisa)
- Menampilkan gambar (disebutnya _boot splash_) di awal nyala sistem
- Bermain permainan _homebrew_

## Di mana bisa mencari aplikasi homebew?

- [Universal-DB](https://db.universal-team.net/ds) ada hampir semua _homebrew_ modern, dengan pencarian dan pengurutan rapih untuk melihat hal terbaru
- [GameBrew](https://www.gamebrew.org/wiki/List_of_all_DS_homebrew) ada banyak _homebrew_ lawas, namun kebanyakan hanya berfungsi di _flashcard_
