# Mencadang DSiWare

## Persyaratan

- Versi terkini dari [GodMode9i](https://github.com/RocketRobz/godmode9i/releases)
  - Unduh arsipnya, ekstrak isinya, dan taruh `GodMode9i.nds` di mana saja di kartu SD

## Nintendo DSi - Instruksi

### Bagian I - Membuat Dump DSiWare

1. Buka GodMode9i
2. Tekan <kbd>START</kbd> untuk membuka Menu START
3. Pilih `Title manager...`
   - Jika tidak muncul, pastikan kartu SD dan NAND **sudah** di-_mount_. Jika dibuka dari hiyaCFW, buka dari tempat lain
4. Pilih judul yang ingin di-_dump_
5. Pilih data untuk di-_dump_
6. Ulangi langkah 4-5 untuk semua DSiWare yang ingin di-_dump_

::: tip

Hasil _dump_ DSiWare akan ada di `sd:/gm9i/out`.

:::
