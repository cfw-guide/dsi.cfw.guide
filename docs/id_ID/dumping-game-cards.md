# Membuat Dump Kartrid

Laman ini untuk men-_dump_ kartrid dengan GodMode9i agar bisa dimainkan di emulator, _flashcard_, atau kartu SD lewat nds-bootstrap.

## Persyaratan

- Kartrid permainan yang ingin di-_dump_
- Konsol Nintendo DSi yang ada [Unlaunch](installing-unlaunch.html)

## Instruksi

### Bagian I - Penyiapan kartu SD

1. Unduh versi terkini dari [GodMode9i](https://github.com/DS-Homebrew/GodMode9i/releases)
2. Ekstrak `GodMode9i.nds` dari arsip GodMode9i, dan taruh di mana saja di kartu SD

### Bagian II - Arahan di Nintendo DSi

1. Buka GodMode9i
2. Pastikan kartrid sudah sisip ke konsol
3. Pilih opsi "KARTRID NDS" di GodMode9i
4. Pilih data untuk di-_dump_
   - Opsi "Dipangkas" akan menghemat ukuran _dump_ ROM di kartu SD, tapi hasil _dump_ ini tidak berfungsi dengan sebagian besar tambalan retasan ROM
5. Ulangi langkah 2-4 untuk semua kartrid yang ingin di-_dump_

::: tip

Hasil _dump_ kartrid akan ada di `sd:/gm9i/out`.

:::
